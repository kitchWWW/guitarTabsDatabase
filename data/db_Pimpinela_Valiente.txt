 (Acorde)    (Cuerdas)
    |       E A D G B E  Los números corresponden al
    |       | | | | | |  traste de la guitarra donde
    |       | | | | | |  se pulsa la cuerda señalada,
    |       | | | | | |  la letra X significa que no
    |       | | | | | |  se pulsa la cuerda indicada.
    |       | | | | | |
    Do   :  X 3 2 0 1 0  ( Do Mayor )
    Do7  :  X 3 2 3 1 0  ( Do Mayor7)
    Do#  :  X 4 6 6 6 4  ( Do Mayor Sostenido )
    Fa   :  1 3 3 2 1 1  ( Fa Mayor )
    Fam  :  1 3 3 1 1 1  ( Fa menor )
    Sol# :  4 6 6 5 4 4  ( Sol Mayor Sostenido )
    La#m :  X 1 3 3 2 1  ( La menor Sostenido )
    La#m7:  X 1 3 1 2 1  ( La menor Sostenido7 )

/  = Slide up
\  = Slide down
h  = Hammer-on
p  = Pull-off
~  = Vibrato
PM = Palm Mute
[] = Tapping harmonic
b?/ = Bend : ? represents how many steps to bend
        bh/   = Bend a 'half' step
        b1/   = Bend a whole step
        b1.5/ = Bend whole and a half steps
b?\ = Release : ? represents how many steps to release.  If no number is
                  present, release it to the original pitch.  This notation
                  could be just '\' as in bh/ \ / for bending, releasing to
                  the original pitch, then bending again
u  = Hold : After bending the previous note, keep the pitch, then pick the
            same string again
        10bh/ 10u = Bend the 10th fret note for a 'h'alf step THEN pick the
                    same string again while keep the half step higher pitch

getting some E-Mails (and listening to the song closely) I must say it is indeed a 
bit tricky. So I tabbed it out. Note that this is still a slightly simplified 
version. The guitarist plays some more bass notes, but if they were in the tab 
that would only confuse, so I left them out. It will sound great if you play it like this.
